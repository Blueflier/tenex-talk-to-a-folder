      - If --dry-run: print question and gold answers, skip API call
      - Otherwise: call query_chat against base_url/chat with session_id=paper_id
      - Score with score_answer(response_text, gold_answers)
      - If score < 1.0: classify failure via classify_failure (need gold answer embedding for RETRIEVAL_MISS check -- embed the best-matching gold answer text)
