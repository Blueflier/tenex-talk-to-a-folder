  <name>Task 1: Backend /reindex endpoint with surgical chunk replacement</name>
  <files>backend/reindex.py, backend/app.py, tests/test_reindex.py</files>
  <behavior>
    - reindex_file drops all chunks with matching file_id, appends new chunks, saves merged session
    - reindex_file calls volume.commit() after save
    - reindex_file calls invalidate_caches(file_id) after save
    - reindex_file returns { file_id, indexed_at } with current UTC ISO timestamp
    - POST /reindex with valid session_id + file_id returns 200 with { file_id, indexed_at }
    - POST /reindex without auth returns 401
    - Other files' chunks are unchanged after reindex
  </behavior>
  <action>
    **Create `backend/reindex.py`:**

    1. `async def reindex_file(user_id, session_id, file_id, access_token)`:
       - Fetch file content from Drive using access_token (reuse Phase 2's file extraction pipeline: `fetch_file_content` + `chunk_file`)
       - Embed new chunks via `embed_chunks` (reuse Phase 2's embedding function)
       - Load existing session: `np.load(f"/data/{user_id}/{session_id}_embeddings.npy")` and `json.load(f"/data/{user_id}/{session_id}_chunks.json")`
       - Build keep mask: `[i for i, c in enumerate(old_chunks) if c["file_id"] != file_id]`
       - Keep chunks/embeddings for other files
       - Merge: `kept_chunks + new_chunks`, `np.vstack([kept_embeddings, new_embeddings])` if keep non-empty else just new_embeddings
       - Save merged session to Volume
       - `volume.commit()`
       - `invalidate_caches(file_id)` from staleness module
       - Return `{"file_id": file_id, "indexed_at": datetime.utcnow().isoformat() + "Z"}`

    2. `POST /reindex` endpoint accepting `{ session_id, file_id }` with Authorization header:
       - Verify token, extract user_id (same auth as /chat)
       - Call `reindex_file(user_id, session_id, file_id, access_token)`
       - Return JSON response with file_id and indexed_at
       - Handle errors: 404 if session not found, 500 on extraction failure

    3. Register route in `backend/app.py`.

    **Create `tests/test_reindex.py`:**
    - test_surgical_replacement: create session with 2 files (A, B), reindex file A, verify file B's chunks unchanged, file A's chunks replaced
    - test_cache_invalidation: after reindex, verify staleness cache and grep text cache cleared for that file_id
    - test_indexed_at_returned: verify response contains file_id and ISO timestamp with Z suffix
    - test_volume_commit_called: mock volume, verify commit() called after save
    - test_reindex_401: no auth header returns 401
  </action>
