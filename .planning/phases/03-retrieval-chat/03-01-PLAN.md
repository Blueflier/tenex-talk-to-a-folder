    Create `backend/chat.py` with:

    2. `stream_llm(query, retrieved_chunks, model_key="deepseek")` — async generator using `get_client(model_key)` from Phase 1's model config. `client.chat.completions.create(stream=True, max_tokens=2000, temperature=0.1)`. Yields delta.content strings.

    3. `POST /chat` endpoint accepting `{ session_id, query, file_list }` with `Authorization: Bearer {token}` header:
       - Verify token, extract user_id (reuse Phase 1 auth verification)
       - Load `{user_id}/{session_id}_embeddings.npy` and `{user_id}/{session_id}_chunks.json` from Modal Volume at `/data`
       - Embed query via OpenAI `text-embedding-3-small`

       - Emit `[DONE]`
       - Return `StreamingResponse(media_type="text/event-stream")`
       - SSE format: `data: {json}\n\n` for each event, `data: [DONE]\n\n` at end
